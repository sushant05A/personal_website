My Projects


Personal expense tracker

     The Personal expense tracker is a web-based application that simplifies expense tracking and management. It allows users to register and log in securely to track their expenses. The system provides a dashboard displaying an overview of expenses, generates detailed reports based on different criteria, and visualizes data through charts and graphs. Users can search, filter, and manage their expenses efficiently. The system also offers profile management, notifications, and reminders. With a responsive design, users can access the system on various devices.

Technologies used -
Programming Language: PHP.
Database: MySQL
HTML (Hypertext Mark-up Language), CSS (Cascading Style Sheets), JavaScript, jQuery (JavaScript library) Bootstrap (CSS framework), Chart.js (JavaScript library for data visualization)
